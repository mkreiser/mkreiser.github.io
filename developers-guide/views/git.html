git

Commands

git add -p - add changes individually
git commit -m "message" - commit with message
git rebase -i HEAD~3 - rebase HEAD interactively with the last 3 commits
git push -u origin localbranch:remotebranch - push remote branch
git log -3 - view last 3 commits
git branch -d (local branch) - delete local branch

Merging Workflow

Note: This is concentrated to web dev, although the general flow applies to all merges.
1.) Check changes in Chrome.
2.) Check changes in Firefox.
3.) Check changes in Safari.
4.) Run tests.
5.) git co master
6.) git pull
7.) git co (branch)
8.) git rebase master
9.) git mergetool if conflicts
10.) Run tests if conflicts
11.) git push --force
12.) Check for code bubbles/issues
13.) Merge branch
14.) Delete branch
