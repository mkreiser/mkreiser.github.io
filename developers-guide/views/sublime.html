Sublime Text 3

Useful User Settings

font_size - set to any pixel size.
highlight_line - set to true to highlight the current line the cursor is on.
save_on_focus_lost - set to true to save files when left.
tab_size - defaults to 4 spaces.
word_wrap - set to true for lines to wrap.
